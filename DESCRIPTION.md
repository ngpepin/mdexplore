# Repository Description

`mdexplore` is a desktop Markdown browser for Ubuntu/Linux built with Python and PySide6.

It provides:

- a directory tree rooted at a chosen path
- fast Markdown preview with caching
- rendering support for MathJax/TeX, Mermaid, and PlantUML
- quick actions for navigating up, editing in VS Code, and quitting
- per-file highlight colors persisted per directory in `.mdexplore-colors.json`
